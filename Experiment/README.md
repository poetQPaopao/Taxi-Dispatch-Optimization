# How to run test

Go to

```./Taxi-Dispatch-Optimization```:

For Mac/Linux:
```bash
bash run_test.sh
```

For Windows:
```bash
run_test.ps1
```

The outputs will be saved to:
```bash
./Taxi-Dispatch-Optimization/outputs
```
